Selector Types

Universal Selectors: This selector will select everything. The universal selector is represented by "*"

Type selectors: This selector will select all elements of this type. It is represented by the name of the element. For example, if I wanted to select all things in paragraph, I would represent it with "p".

Class Selectors: From what I understand, a class is an attribute that I can make up for organization. Class selectors let you select everything in that class. It is represented by: .classname. This does not work if the class starts with a number. You add multiple classes to an element by separating with spaces.

ID Selectors: This is like a class selector but you each element can only have one id selector and it can only appear once per page. Use sparringly if at all. These cannot start with a number. It is represented by #idname

The Grouping Selector: This selector is used to reduce repition. Say I have two elements and I wanted to have somethings about them the same while I wanted other things to be different. I use the grouping selector to acheive this. The selector is .firstclass, .secondclass.

Chaining Selectors: This is a way to be specific. You can select elements if they have all of the classes. These are represented by .class1.class2. This will select everything that has class 1 and 2. You can also use .class1#class2. to chain a class selector to an id selector. You cannot chain more than one type of type selectors.

Descendant Combinator:Allows me to select an element if it also has the anscestor I am asking for. They are represented by .ancestor .child. Note the space. combine
